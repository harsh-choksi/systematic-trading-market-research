## Strategy Notes

TREND-CATCHER [15m] uses trend-state detection, stochastic oscillator confirmation, dynamic thresholds, and rule-based trade management. The repository keeps the active script at the root so reviewers can find the current implementation quickly.

## Backtest Caveats

Backtested results depend on TradingView settings, selected symbols, timeframe, commissions, slippage, historical data quality, and strategy assumptions. They should be treated as research output rather than live trading expectations.
